The purpose of this plugin is to improve business productivity for small businesses and organizations especially 
those with employees and contractors that telecommute or work remotely. This plugin helps you to set and maintain 
operational structure, assists with process flow, and encourages workers to remain on task, all which lead to improved 
results.

== Description ==

B-Productiv Lite is a free productivity improvement plugin that helps improve productivity and efficiency.  
Built to integrate seamlessly with WordPress, you can set up tasks, alerts, step-by-step procedures, reminders, schedules, 
and their priority levels. The B-Productiv plugin will assist you by making sure these tasks are done. In essence, 
it is like the perfect office manager with the same passion for your business that you have, with exceptional organizational skills. 
It frees you up from many of the everyday indirect operations within your organization, improves communication among your team, 
and allows you to get back to the core tasks of your organization. Even more……it helps you get back to making MONEY.



== Installation ==

This section describes how to install the plugin and get it working.
Before installing, you should backup your MySQL database and website.

= Automatic installation =

1. Log in to your WordPress dashboard.
2. Navigate to the "Plugins" menu on the left side of the screen.
3. Click Add New.
4. In the search field type "B-Productiv" and click Search Plugins.  
5. Install it by simply clicking "Install Now".
6. Activate the plugin through the "Plugins" screen in WordPress.

= Manual installation =

1. Download the B-productiv plugin from Wordpress.org.
2. Install the B-Productiv plugin through the WordPress plugins screen directly by navigating to "Plugins" 
located in the list of tabs on the left side of the screen.
3. Click "Add New".
4. Click "Upload Plugin" and select the B-Productiv plugin file.
5. Activate the plugin through the "Plugins" screen in WordPress.

= Configuration =

Use the B-Productiv screen to configure the plugin. The B-Productiv screen can be accessed
clicking "B-Productiv" in the list of tabs on the left side of the screen. 
	
1. Select a Super Admin from the drop down menu. This admin will be the only person allowed to see and change B-Productiv settings.
2. Choose a city or country in your timezone.
3. Select the date format for the plugin.
4. Indicate whether you want data saved in the database by the B-Productiv Lite plugin to be saved when the plugin is deleted.
5. Click "Save Settings" when finished.
6. Use the "Link to the Portal" to proceed to the Portal.
